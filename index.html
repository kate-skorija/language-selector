You should learn Ruby!

Ruby is an interpreted, multipurpose programming language. It is very popular with start-ups and small businesses, but is also behind large companies like AirBnb. Ruby on Rails is a very popular backend framework written Ruby.

You should learn C#!

C# can be used for many different tasks, but is particularly great at building Windows desktop applications and games. C# is typically used by large corporations such as Microsoft.

You should learn JavaScript!

JavaScript is arguably the most popular and widespread programming language. It is an essential tool in your programming toolbelt! jQuery and React are widely-used JavaScript libraries.
